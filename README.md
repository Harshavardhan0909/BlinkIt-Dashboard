Blinkit Sales Analytics Dashboard 📊

🟡 Project Title
Blinkit Sales Analytics Dashboard using Power BI

📌 Project Overview

This project is an interactive Power BI Dashboard developed for analyzing Blinkit sales performance, outlet operations, customer ratings, and product distribution insights. The dashboard helps stakeholders understand business trends, identify high-performing outlet categories, and make data-driven decisions.

The dashboard provides a complete overview of:

Total Sales
Average Sales
Number of Items
Average Ratings
Outlet Performance
Product Category Analysis
Outlet Size & Location Insights
❓ Business Problem

Blinkit operates across multiple outlet types, sizes, and locations with thousands of products. Managing and analyzing sales performance manually becomes difficult due to large datasets and multiple KPIs.

The business needs a centralized analytics dashboard to:

Track overall sales performance
Identify top-performing item categories
Compare outlet types and outlet sizes
Analyze customer ratings
Understand sales distribution across locations
Improve operational and marketing decisions

This Power BI dashboard solves the problem by converting raw sales data into meaningful visual insights.

🎯 Objectives
Analyze Blinkit sales performance across different outlets
Monitor KPIs like sales, ratings, and item visibility
Compare sales by outlet type, size, and location
Identify high-demand product categories
Build an interactive and user-friendly dashboard
🛠️ Tech Stack
Power BI – Data Visualization & Dashboard Creation
Microsoft Excel / CSV – Data Source
